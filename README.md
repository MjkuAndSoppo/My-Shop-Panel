\# MyShopPanel

轻量级 Minecraft 1.20.1 Forge 经济模组。 

\# 功能 

\- MSPP 点数系统 — 独立货币，支持负数余额（透支/欠款） 

\- 玩家自由市场 — 上架物品、浏览挂单、购买交易

\- OP 官方商店 — 管理员可配置买入/卖出商店条目

\- 报价终端 — 统一入口物品，所有功能集中于此

\# 设计原则 

\- 独立 GUI

\- 继承 Screen 自绘界面，脱离原版容器系统，零冲突

\- 三步提交 — Prepare → Confirm → Commit，事务落盘，取消/超时自动回滚

\- 透支明确告知 — 涉及负余额操作前弹出高亮警告
